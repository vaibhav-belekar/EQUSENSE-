## 🔍 Detailed Data Sources

### 1. **Stock Price Data** (Yahoo Finance)
- **Source**: Yahoo Finance via `yfinance`
- **What it provides**:
  - Open, High, Low, Close prices
  - Volume
  - Historical data (1 day to max)
- **How to access**: 
  - Automatically fetched when you run a trading cycle
  - Cached in `data_collector.data_cache`

### 2. **Technical Indicators** (Calculated Locally)
- **Source**: Calculated from price data using `ta` library
- **Indicators**:
  - RSI (14-period)
  - MACD (12, 26, 9)
  - Bollinger Bands (20-period)
  - SMA 20, SMA 50
  - EMA 12, EMA 26
  - ATR (14-period)

### 3. **ML Model Predictions** (LSTM)
- **Source**: Trained on historical data
- **Input**: Last 60 days of price + indicators
- **Output**: Up/Down/Neutral signal with confidence

### 4. **Portfolio Data** (Simulated)
- **Source**: Generated by Trader Agent
- **What it tracks**:
  - Current holdings
  - Cash balance
  - Trade history
  - Portfolio value

## 📡 Data Fetching Process

### When Data is Fetched

1. **On Ecosystem Initialization**
   ```python
   ecosystem.initialize()  # Fetches 3 months of data
   ```

2. **On Trading Cycle**
   ```python
   ecosystem.run_cycle()  # Fetches latest data for predictions
   ```

3. **On Frontend Request**
   ```javascript
   // Frontend calls backend API
   GET /api/predictions  // Backend fetches fresh data
   ```

### Data Caching

- Data is cached in `DataCollector.data_cache`
- Reduces API calls to Yahoo Finance
- Cache is per symbol and session

## 🔄 Real-Time Data Flow

### Step-by-Step Data Journey

1. **User clicks "Run Cycle" in frontend**
   ↓
2. **Frontend sends POST to `/api/run-cycle`**
   ↓
3. **Backend calls `ecosystem.run_cycle()`**
   ↓
4. **Analyst Agent calls `data_collector.fetch_data()`**
   ↓
5. **yfinance fetches from Yahoo Finance**
   ↓
6. **DataCollector calculates indicators**
   ↓
7. **Analyst Agent makes predictions**
   ↓
8. **Trader Agent makes decisions**
   ↓
9. **Results sent back to frontend**
   ↓
10. **Frontend displays updated data**

## 📊 What Data is Displayed in Frontend

### Portfolio Overview
- **Source**: `TraderAgent.portfolio` + current prices
- **Updated**: After each trading cycle

### Trading Charts
- **Source**: `AuditorAgent.performance_history`
- **Data**: Portfolio value over time

### Predictions
- **Source**: `AnalystAgent.predictions`
- **Data**: Stock signals and confidence scores

### Trade History
- **Source**: `TraderAgent.trade_history`
- **Data**: All executed trades

### Performance Metrics
- **Source**: `AuditorAgent.evaluate_performance()`
- **Data**: Win rate, Sharpe ratio, drawdown, etc.

## 📝 Data Storage

- **No database**: Data is in-memory during session
- **Cache**: Stored in `DataCollector.data_cache`
- **History**: Stored in `AuditorAgent.performance_history`
- **Trades**: Stored in `TraderAgent.trade_history`

## ⚠️ Important Notes
