# Trading Ecosystem Frontend

Modern React frontend for the Multi-Agent Trading Ecosystem.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Features

- 🎨 Modern, responsive UI
- 📊 Interactive charts
- 🤖 Real-time agent status
- 📈 Portfolio tracking
- 🔄 Auto-refresh capability

See `FRONTEND_SETUP.md` for detailed setup instructions.
